Easily create page loading 

    Example:
    <div class="loading light-1">   <img src="images/loading.svg"> <div>
    
    Background Color Opacity options: 
    light-1 -40% Opacity
    light-2 -60% Opacity
    light-3 -90% Opacity

    <img> is optional for you. You can change it. You can use, animated gif, icon, image etc...
    <div class="loading light-1">   <img src="images/loading.svg">  <div>


    Example with caption:
    <div class="loading light-1"> <img src="images/loading.svg"> <div> Loading... </div> <div>
