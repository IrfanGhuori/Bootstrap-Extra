Set background image 

  Easily set web background image just class='img-cover'  into the <img src=" " >
  Example:  <img class="img-cover " src="images/bg.jpg " alt="web-cover ">

  (Optional)
  Add Pattern on the background image 9 option available. 
  
  pattern-1 - increase  10%
  pattern-2 - increase  30%
  pattern-3 - increase  50%
  pattern-4 - increase  70%
  pattern-5 - increase  90%
  pattern-6 - increase  110%
  pattern-7 - increase  130%
  pattern-8 - increase  150%
  pattern-9 - increase  200%

  Code Example:

  <div class="pattern-1 "></div>
  <img class="img-cover " src="images/bg.jpg " alt="web-cover "">
